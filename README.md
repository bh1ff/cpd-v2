# Robocode × LIA — Digital Youth Work CPD

A practical, discussion-heavy 6-session CPD programme for youth practitioners.

## What's here

- **6 session pages** — full lesson plans with LIA template, 10–13 activities each, and D-A-K-D-C phase structure
- **12 downloadable files** — PowerPoint slides + Word facilitator guides for every session
- **How to Use** — facilitation guide for delivering the programme

## Sessions

1. **What Is Digital Youth Work?** — Environment design as practice (Kahoot-based)
2. **Your Values Under Pressure** — Stated values vs observable behaviour (Padlet/Mentimeter)
3. **Keeping Young People Safe Online** — Three contexts: public, private, DM
4. **Social Media and Engagement** — Motivation vs manipulation in platform mechanics
5. **Where Are Young People Online?** — Live platform audits (5-criteria framework)
6. **Build Something Real** — Peer-tested digital intervention

## Stack

Static HTML + CSS. No build step. Drop it on any static host (GitHub Pages, Netlify, Vercel).

## Local preview

```
python3 -m http.server -d . 8000
```
